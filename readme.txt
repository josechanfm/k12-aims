YEAR
1. New Academic start
新學年必須先建立,才能進行所有涉及下一學年的操作.比方說學生升班.
而在建立新學年的同時,年級(Grade)和班別(Klass)也必須同時建立.否則升班的年級和班別會受到影響.

年級分為幼稚園、小學和中學三年級段,不建議分為初中、高中或初小、高小.而新學年與年級、班別一並建立後,也可以一個學年的對年級和班別進行管理,包括新增、刪除或修改,但必需適合數據庫的一致原則.

學年數據庫包括當前學年的標記欄位(Active),表示當前學年的年份,也就是說,需要進行學年轉換的操作.其他功能同一以當前年份為默認值.


注意:
學年的開始與結束日是非常關鍵,並不需要準確的月與日,但年份符合學年的順序.因為升班是以學年為基礎,從所屬的學年進升到以學年開始日計算的下一個學年.
